# plesk.domains.windows
Use the parkingomat.plesk module on windows environment


# First Steps


## on windows

### install
    .apibuild\\install.bat

### start
    .apiexec\\start.bat

### open in browser
    .apicra\\browser.bat



## on linux

### install
    sh .apibuild/install

### start
    sh .apiexec/start

### open in browser
    sh .apicra/browser
