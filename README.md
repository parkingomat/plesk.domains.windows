# plesk.domains.windows
Use the parkingomat.plesk module on windows environment


# First Steps


## on windows

### install
    .apibuild\\install.bat

### install
    .apifunc\\download.bat
    .apifunc\\delete.bat

### start
    .apiexec\\start.bat

### open in browser
    .apicra\\browser.bat
